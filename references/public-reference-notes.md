# Public Reference Notes

This project uses public information only. No confidential utility drawings, internal standards, customer information, or Toronto Hydro project records are used.

## Reference Areas

The project is inspired by publicly visible utility design and construction workflows, including:

- CAD-based infrastructure drawing production
- Electrical/civil plan drawings
- Plan and profile drawing organization
- Single-line diagram documentation
- Voltage drop calculation workflows
- Cable pulling review
- Field verification checklists
- Project record tracking

## Notes

The design assumptions used in this project are simplified for academic and portfolio purposes. Final utility designs require review by qualified professionals and the relevant utility/design authority.
