# Project Scenario

## Project Title

Electrical Distribution Design Package Using MicroStation and Excel

## Purpose

This project simulates a utility style design package for a fictional commercial customer service connection. The goal is to build practical familiarity with MicroStation drafting, electrical/civil drawing documentation, voltage drop calculations, cable pulling review, field verification planning, and project record tracking.

## Fictional Site Information

- Project Name: Mock Commercial Customer Service Connection
- Location: 1234 Rexdale Industrial Road, Toronto, ON
- Customer Type: Small commercial warehouse
- Service Voltage: 600 V, 3-phase
- Estimated Load: 250 kVA
- Main Service Size: 400 A
- Cable Route Length: 85 m
- Transformer: Existing padmount transformer
- Proposed Work: Underground secondary service duct bank from existing transformer to customer service entrance

## Proposed Design Scope

The proposed design includes:

- Existing conditions plan
- Proposed electrical/civil service route
- Underground duct bank route
- Pull box location
- Plan and profile drawing
- Single-line diagram
- Construction detail sheet
- Voltage drop calculation
- Cable pulling review
- Quantity takeoff
- Project record tracker
- Field verification checklist
- Training slide deck

## Important Disclaimer

This is an academic portfolio project created for learning purposes only. It is not affiliated with Toronto Hydro and is not intended for construction.
