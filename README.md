# Electrical Distribution Design Package Using MicroStation and Excel

This project simulates a utility-style electrical distribution design package for a fictional commercial customer service connection in Toronto. The objective is to develop practical experience with MicroStation drafting, electrical and civil drawing documentation, voltage drop calculations, cable pull review, quantity takeoffs, project record management, field verification planning, and technical reporting.

---

## Project Scenario

A fictional commercial warehouse requires a new 600 V, 3-phase underground secondary service connection from an existing padmount transformer to a customer service entrance. The proposed service route consists of an 85 m underground duct bank, one pull box, and supporting engineering documentation including plan drawings, calculations, construction details, and project records.
